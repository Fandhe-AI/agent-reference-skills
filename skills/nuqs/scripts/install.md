# Install

nuqs パッケージのインストールコマンド集。

## npm でのインストール

```sh
npm install nuqs
```

## pnpm でのインストール

```sh
pnpm add nuqs
```

## yarn でのインストール

```sh
yarn add nuqs
```

## bun でのインストール

```sh
bun add nuqs
```

## 旧バージョン（nuqs v1）のインストール

Next.js の古いバージョンを使用する場合は v1 系を指定する。

```sh
npm install nuqs@^1
```

```sh
pnpm add nuqs@^1
```
