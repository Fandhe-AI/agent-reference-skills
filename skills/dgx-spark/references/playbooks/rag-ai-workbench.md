# RAG Application in AI Workbench

Deploy a pre-built agentic RAG application in NVIDIA AI Workbench that routes queries, evaluates relevancy/hallucination, and iterates generation cycles via a Gradio UI.

## Signature / Usage

```bash
git clone https://github.com/NVIDIA/workbench-example-agentic-rag
# configure project secrets: NVIDIA API key, Tavily API key
# launch Gradio chat interface from AI Workbench
```

## Notes

- Requires NVIDIA AI Workbench installed, free NVIDIA API key (Public API Endpoints permission), free Tavily API key
- Supports NVIDIA-hosted API endpoints or self-hosted models
- Duration: 30-45 minutes; Risk: Low
