# DGX OS

Customized Linux distribution designed specifically for DGX systems, providing a stable, tested, and supported operating system foundation for running AI, machine learning, and analytics applications.

## Key Features

- Linux foundation optimized for AI workloads
- Pre-configured drivers and system settings tailored for NVIDIA hardware
- Security updates and system maintenance tools
- Integration with NVIDIA's broader software ecosystem

## Security and Compliance

DGX OS is built on Ubuntu and leverages its security capabilities, referencing Ubuntu security standards for supported compliance frameworks including FIPS, CIS, and DISA-STIG.

## Release Cadence

Major updates typically arrive twice annually (around February and August) during the first two years following initial release. Security patches and interim updates are provided throughout the support lifecycle.

## Notes

- DGX Spark recovery differs from enterprise DGX systems: do not use the Enterprise Download Center or DGX OS ISO. Recovery images for DGX Spark are available from developer.nvidia.com without requiring an enterprise account.
- Full documentation: [DGX OS User Guide](https://docs.nvidia.com/dgx/dgx-os-7-user-guide/introduction.html)
