# NVIDIA AI Enterprise Quick Start

Enterprise-grade software platform for AI development, deployment, and optimization on DGX Spark systems, including libraries, frameworks, microservices, and enterprise support.

## Initial Setup

Complete basic DGX Spark system configuration first, including powering on and connecting to a network. See the [First Boot guide](../setup/first-boot.md).

## Trial Option

A 90-day evaluation is available through [NVIDIA's Enterprise Product Registration](https://enterpriseproductregistration.nvidia.com/?LicType=Eval&ProductFamily=NVAIE_DGX_SPARK).

## Account Creation and Access

1. **Receive confirmation**: NVIDIA sends order details and an Entitlement Certificate
2. **Register**: follow the certificate link and complete the registration form
3. **Access hub**: log into the [NVIDIA Application Hub](http://nvid.nvidia.com/dashboard) using the emailed credentials
4. **Set password**: create account credentials and verify the email address

## Accessing Software

Navigate to the [NVIDIA NGC Catalog](https://catalog.ngc.nvidia.com/), apply the DGX Spark filter or find the DGX Spark Collection, then download the desired software assets.

## Notes

- Account access enables two key services: NVIDIA NGC (software access) and the NVIDIA Enterprise Support Portal.
