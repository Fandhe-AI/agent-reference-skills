# Software Overview

DGX Spark ships with a comprehensive software stack optimized for AI development, machine learning, and data science workflows.

## Components

| Component | Description |
| --- | --- |
| DGX Dashboard | Built-in web dashboard for system metrics, updates, configuration, and integrated JupyterLab |
| NVIDIA Sync | System tray utility to manage DGX Spark and other remote Linux systems from a local computer |
| DGX OS | Customized Ubuntu-based Linux distribution for AI, machine learning, and analytics workloads |
| NVIDIA Nsight | GPU profiling, debugging, and analysis tools for CUDA and graphics workloads |
| NVIDIA Container Runtime for Docker | Enables Docker containers to access GPU resources |
| NGC | NVIDIA GPU Cloud — GPU-optimized containers, pre-trained models, and AI/ML software |
| NVIDIA AI Enterprise | Enterprise-grade software platform for AI development, deployment, and optimization |
