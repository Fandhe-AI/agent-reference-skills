# Commands

syncpack CLI コマンド一覧。

| Name | Description | Path |
|------|-------------|------|
| fix | Autofix dependency version mismatches across your monorepo to match configured rules. | [fix.md](./fix.md) |
| format | Sort `package.json` fields into a consistent order and alphabetize nested fields across all packages. | [format.md](./format.md) |
| json | Export all dependency instance data from a monorepo as newline-delimited JSON (one object per instance). | [json.md](./json.md) |
| lint | Check for dependency version mismatches across a monorepo. Reports issues without modifying files. | [lint.md](./lint.md) |
| list | Display all dependencies across a monorepo with their versions and locations. | [list.md](./list.md) |
| update | Update dependencies in your monorepo to newer versions from the npm registry. | [update.md](./update.md) |
