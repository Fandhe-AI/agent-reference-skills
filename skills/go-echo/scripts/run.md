# run

Run and build an Echo application with the standard Go toolchain.

## Run the server

```sh
go run main.go
```

Executes the quickstart "Hello, World!" server; by default it listens on `http://localhost:1323`.

## Build a binary

```sh
go build
```

Compiles the current module into an executable binary named after the module/directory.

## Build with an explicit output name

```sh
go build -o myapp
```

## Run the compiled binary

```sh
./myapp
```
