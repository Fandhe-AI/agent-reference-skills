# Examples

Kubb の各プラグインを使った kubb.config.ts の設定例集。

| Name | Description | Path |
|------|-------------|------|
| Advanced Configuration Example | 複数プラグインを組み合わせた高度な設定例。TypeScript 型、API クライアント、React Query hooks、Zod スキーマ、MSW ハンドラーを同時に生成する。 | [advanced.md](./advanced.md) |
| Axios Client Example | OpenAPI 仕様から Axios ベースの API クライアントを生成する設定例。 | [client.md](./client.md) |
| Cypress Example | OpenAPI 仕様から Cypress リクエスト定義を生成する設定例。 | [cypress.md](./cypress.md) |
| Custom Generators Example | カスタムジェネレーターを使って生成コードをカスタマイズする設定例。 | [generators.md](./generators.md) |
| Faker Example | OpenAPI 仕様から Faker.js モックデータジェネレーターを生成する設定例。 | [faker.md](./faker.md) |
| Fetch Client Example | OpenAPI 仕様から Fetch ベースの API クライアントを生成する設定例。 | [fetch.md](./fetch.md) |
| MCP Example | OpenAPI 仕様から MCP（Model Context Protocol）サーバーを生成する設定例。 | [mcp.md](./mcp.md) |
| MSW Example | OpenAPI 仕様から MSW（Mock Service Worker）ハンドラーを生成する設定例。 | [msw.md](./msw.md) |
| React Query Example | OpenAPI 仕様から React Query（TanStack Query）hooks を生成する設定例。 | [react-query.md](./react-query.md) |
| Simple Example | 最小限の Kubb 設定例。OpenAPI 仕様から基本的なコードを生成する。 | [simple.md](./simple.md) |
| Solid Query Example | OpenAPI 仕様から Solid Query hooks を生成する設定例。 | [solid-query.md](./solid-query.md) |
| Svelte Query Example | OpenAPI 仕様から Svelte Query hooks を生成する設定例。 | [svelte-query.md](./svelte-query.md) |
| SWR Example | OpenAPI 仕様から SWR hooks を生成する設定例。 | [swr.md](./swr.md) |
| TypeScript Example | OpenAPI 仕様から TypeScript 型を生成する設定例。複数の enum スタイルと出力形式を示す。 | [typescript.md](./typescript.md) |
| Vue Query Example | OpenAPI 仕様から Vue Query hooks を生成する設定例。 | [vue-query.md](./vue-query.md) |
| Zod Example | OpenAPI 仕様から Zod バリデーションスキーマを生成する設定例。 | [zod.md](./zod.md) |
