# New Releases

BullMQ Pro の新バージョンリリース通知の取得方法。

## Notes

リリース通知を受け取るには、GitHub の追跡 Issue (`bullmq-pro-support` リポジトリ Issue #86) で通知を有効にする。新リリースごとに自動コメントが投稿される。

完全な変更履歴は [Changelogs](https://api.bullmq.pro) を参照すること。
