# nestjs-bullmq-pro Changelog

`nestjs-bullmq-pro` パッケージの変更履歴。

## Notes

- v3.1.0 (2025-03-06): NestJS v11 サポートを追加
- v3.0.0 (2024-04-05): bullmq-pro v7 サポートを追加（破壊的変更あり。移行ガイドは "Better Queue Markers" および "BullMQ Pro v7 Release" を参照）
- v2.1.0: bullmq-pro を v7.4.0 に更新
- v2.0.0 (2023-06-28): NestJS v10 および bullmq-pro v6 サポート追加。NestJS v7 サポートを終了
- v1.1.0: FlowProducer Pro サポートを追加
- v1.0.0 (2023-01-XX): 初版リリース、WorkerPro 統合

完全な変更履歴は [API Reference](https://nestjs.bullmq.pro/) を参照すること。
