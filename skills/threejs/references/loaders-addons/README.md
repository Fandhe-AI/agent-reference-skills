# Loaders (Addons)

Addon loader classes imported from `three/addons/loaders/`. Not included in the core `three` bundle.

| Name | Description | Path |
|------|-------------|------|
| GLTFLoader | Load glTF 2.0 models — the recommended 3D asset format | [GLTFLoader.md](./GLTFLoader.md) |
| DRACOLoader | Load Draco-compressed geometry (`.drc`); integrates with GLTFLoader | [DRACOLoader.md](./DRACOLoader.md) |
| FBXLoader | Load FBX 3D models (ASCII ≥ 7.0, binary ≥ 6400) | [FBXLoader.md](./FBXLoader.md) |
| OBJLoader | Load Wavefront OBJ models | [OBJLoader.md](./OBJLoader.md) |
| MTLLoader | Load MTL material files; used together with OBJLoader | [MTLLoader.md](./MTLLoader.md) |
| FontLoader | Load typeface JSON fonts for TextGeometry | [FontLoader.md](./FontLoader.md) |
| SVGLoader | Load SVG files and convert paths to Three.js Shapes | [SVGLoader.md](./SVGLoader.md) |
| EXRLoader | Load OpenEXR HDR textures | [EXRLoader.md](./EXRLoader.md) |
| HDRLoader | Load RGBE HDR textures (`.hdr`) for environment maps | [HDRLoader.md](./HDRLoader.md) |
| KTX2Loader | Load KTX 2.0 Basis Universal GPU-compressed textures | [KTX2Loader.md](./KTX2Loader.md) |
| STLLoader | Load STL geometry (binary and ASCII) | [STLLoader.md](./STLLoader.md) |
