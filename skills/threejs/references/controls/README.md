# Controls

| Name | Description | Path |
|------|-------------|------|
| OrbitControls | Orbit/zoom/pan around a target point; maintains camera up direction. Most commonly used control | [OrbitControls.md](./OrbitControls.md) |
| ArcballControls | Virtual trackball control with gizmo, animations, and state serialization; no `update()` needed | [ArcballControls.md](./ArcballControls.md) |
| DragControls | Makes 3D objects draggable with pointer interaction; fires drag/hover events | [DragControls.md](./DragControls.md) |
| FirstPersonControls | First-person walk-through camera with mouse-look and keyboard movement | [FirstPersonControls.md](./FirstPersonControls.md) |
| FlyControls | Free-form 6-DOF fly navigation with no fixed target | [FlyControls.md](./FlyControls.md) |
| MapControls | OrbitControls subclass with left-mouse-pan default for map-style top-down navigation | [MapControls.md](./MapControls.md) |
| PointerLockControls | First-person control using Pointer Lock API for game-style mouse capture | [PointerLockControls.md](./PointerLockControls.md) |
| TrackballControls | Trackball rotation without maintaining camera up; full-sphere rotation | [TrackballControls.md](./TrackballControls.md) |
| TransformControls | Translate/rotate/scale objects in the viewport with a gizmo UI | [TransformControls.md](./TransformControls.md) |
