# Lights

| Name | Description | Path |
|------|-------------|------|
| Light | Abstract base class for all light types | [Light.md](./Light.md) |

| PointLight | Omnidirectional point-source light; can cast shadows | [PointLight.md](./PointLight.md) |
| RectAreaLight | Uniform rectangular area light; PBR only, no shadows | [RectAreaLight.md](./RectAreaLight.md) |
| SpotLight | Cone-shaped directional light; can cast shadows | [SpotLight.md](./SpotLight.md) |
