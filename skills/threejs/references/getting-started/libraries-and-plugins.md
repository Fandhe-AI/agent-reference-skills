# Libraries and Plugins

Community-maintained list of compatible external libraries and plugins for three.js.

## Notes

- This list is not guaranteed to be up to date; always check the linked projects for current compatibility.

## Categories

### Physics
- **Oimo.js** — Lightweight physics engine
- **enable3d** — Physics integration layer
- **ammo.js** — JavaScript port of Bullet Physics
- **cannon-es** — Physics engine
- **rapier** — Fast physics simulation
- **Jolt** — Physics engine for JavaScript

### Postprocessing
- **postprocessing** — Additional effects and rendering pipeline beyond official three.js postprocessing

### Intersection / Raycast Performance
- **three-mesh-bvh** — BVH-accelerated raycasting and intersection

### Path Tracing
- **three-gpu-pathtracer** — GPU-accelerated path tracing

### File Formats
- **urdf-loader** — URDF file support
- **3d-tiles-renderer-js** — 3D Tiles format
- **WebWorker OBJLoader** — Multi-threaded OBJ loading
- **IFC.js** — IFC (Industry Foundation Classes) format

### Geometry
- **THREE.MeshLine** — Advanced line geometry

### 3D Text and Layout
- **troika-three-text** — High-quality antialiased text
- **three-mesh-ui** — 3D UI layout system

### Particle Systems
- **three.quarks** — Particle system
- **three-nebula** — Advanced particle effects

### Inverse Kinematics
- **THREE.IK**, **fullik**, **closed-chain-ik** — IK solvers

### Game AI
- **yuka** — Steering, pathfinding, decision-making
- **three-pathfinding** — Navigation utilities
- **recast-navigation-js** — Navigation mesh and pathfinding

### Wrappers and Frameworks
- **A-Frame** — Entity-component-system framework
- **react-three-fiber** — React renderer for three.js
- **Threlte** — Svelte components for three.js
- **tresjs** — Vue components for three.js
- **Lume** — HTML custom elements for 3D
- **Needle Engine** — Integrated 3D framework
- **Giro3D** — Geospatial data visualization
- **Mattercraft** — Browser-based AR/WebXR visual editor
