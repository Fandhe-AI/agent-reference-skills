# Install

Three.js および関連パッケージのインストール

## Three.js のインストール

```sh
npm install --save three
```

## TypeScript 型定義のインストール

```sh
npm install --save-dev @types/three
```

## Vite をビルドツールとしてインストール

```sh
npm install --save-dev vite
```

## Three.js + Vite を一括インストール（新規プロジェクト向け）

```sh
npm install --save three && npm install --save-dev vite
```
