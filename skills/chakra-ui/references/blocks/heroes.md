# Heroes

Bold, engaging hero sections designed to showcase your most compelling content.

> **Note:** Blocks require `CHAKRA_PRO_API_KEY` environment variable. If not set, block features are unavailable.

## Group

marketing

## When to Use

- ランディングページのファーストビュー（ヒーローセクション）を構築するとき
- 画像・動画付きの大型ヘッダーセクションが必要なとき
- 検索バー付きのヒーローエリアを作るとき
- サインアップフォーム付きのヒーローが必要なとき
- CLI コマンド表示付きのプロダクトヒーローを作るとき

## MCP Parameters

Tool: `get_component_templates`

| Parameter | Type | Description |
|-----------|------|-------------|
| `category` | `string` | 固定値: `"heroes"` |
| `id` | `string` | 取得するバリアントの ID |
