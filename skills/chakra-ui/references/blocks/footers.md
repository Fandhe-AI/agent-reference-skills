# Footers

Essential bottom sections featuring navigation, contact details, and vital site-wide information.

> **Note:** Blocks require `CHAKRA_PRO_API_KEY` environment variable. If not set, block features are unavailable.

## Group

marketing

## When to Use

- サイトフッターを構築するとき
- 複数カラムのリンク付きフッターが必要なとき
- SNS リンク付きフッターを作るとき
- 住所・連絡先付きのフッターが必要なとき

## MCP Parameters

Tool: `get_component_templates`

| Parameter | Type | Description |
|-----------|------|-------------|
| `category` | `string` | 固定値: `"footers"` |
| `id` | `string` | 取得するバリアントの ID |

## Variants

| ID | Name | Access |
|----|------|--------|
| `footer-with-address` | Footer With Address | paid |
| `footer-with-four-columns` | Footer With Four Columns | paid |
| `footer-with-links-centered` | Footer With Links Centered | paid |
| `footer-with-logo-below` | Footer With Logo Below | paid |
| `footer-with-social` | Footer With Social | free |
