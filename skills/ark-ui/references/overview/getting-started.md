# Getting Started

Install a framework-specific Ark UI package and import components directly, e.g. `Slider` from `@ark-ui/react/slider`.

## Signature / Usage

```tsx
// 1. Install (React example)
// npm install @ark-ui/react

// 2. Import and use a component
import { Slider } from '@ark-ui/react/slider'
```

## Notes

- Package name shown above is for React (`@ark-ui/react`); other frameworks use their own package. See [Framework Differences](./framework-differences.md) for the full list and syntax differences.
- Ark UI is a headless component library — it does not include default styles. Target components using `data-scope` and `data-part` attributes for custom styling.
- StackBlitz quickstart templates are available for Next.js, Solid Start, Nuxt, and Svelte to bypass manual setup.
- Support channels: GitHub repository issues and Discord community.
