# About Ark UI

Ark UI is a headless component library that provides 40+ components and tools for building accessible user interfaces across multiple JavaScript frameworks.

## Signature / Usage

```tsx
import { Slider } from '@ark-ui/react/slider'
```

## Notes

- Supports React, Solid, Vue, and Svelte simultaneously, rather than being tied to a single ecosystem.
- Built on top of [Zag.js](https://zagjs.com), which uses Finite State Machines (FSM) as its foundation. Components are framework-agnostic at the core, with lightweight framework-specific adapters handling events and state management.
- Ark UI reportedly outperformed other headless component libraries in benchmarks despite supporting multiple frameworks.
- Maintained by the Chakra UI team (Segun Adebayo, creator of Chakra UI and Zag.js, alongside Christian Busch and Esther Agbaje).
- Inspired by Park UI, Radix Vue, and the broader open-source community.
- MIT licensed.
