# install

Install a framework-specific Ark UI package with your preferred package manager.

## Install for React

```sh
npm install @ark-ui/react
```

```sh
pnpm install @ark-ui/react
```

```sh
yarn add @ark-ui/react
```

```sh
bun add @ark-ui/react
```

## Install for Vue

```sh
npm install @ark-ui/vue
```

```sh
pnpm install @ark-ui/vue
```

```sh
yarn add @ark-ui/vue
```

```sh
bun add @ark-ui/vue
```

## Install for Svelte

```sh
npm install @ark-ui/svelte
```

```sh
pnpm install @ark-ui/svelte
```

```sh
yarn add @ark-ui/svelte
```

```sh
bun add @ark-ui/svelte
```

Import components via their subpath (e.g. `import { Slider } from '@ark-ui/react/slider'`). Ark UI is a headless library with no default styles, so target parts using `data-scope` / `data-part` attributes.
