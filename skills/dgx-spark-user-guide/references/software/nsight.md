# NVIDIA Nsight

Family of GPU profiling, debugging, and analysis tools for CUDA and graphics workloads. DGX Spark integrates these tools following the broader NVIDIA software release model, with current installers and documentation maintained on the NVIDIA Developer portal.

## Components

| Tool | Description |
| --- | --- |
| Nsight Systems | Timeline-based profiling across CPU, GPU, and system resources during full application execution |
| Nsight Graphics | Frame capture and pipeline inspection for graphics APIs |
| Nsight DL Designer | Visualization and analysis for deep learning model assessment |
| Nsight Compute | CUDA kernel profiling, memory performance evaluation, and guided optimization reports |

## Notes

- Each Nsight tool maintains dedicated developer pages for downloading current builds and accessing documentation, release notes, and installation guidance on the NVIDIA Developer website.
