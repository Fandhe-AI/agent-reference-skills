# Telemetry

| Name | Description | Path |
|------|-------------|------|
| ログドレイン | ログを外部サービスにリアルタイムで転送する機能。Datadog / Loki / Sentry / Axiom / S3 対応。 | [log-drains.md](./log-drains.md) |
| ログ確認 | Supabase ダッシュボードで各サービスのログを確認。API / Auth / Database / Storage / Edge Functions ログ。 | [logs.md](./logs.md) |
