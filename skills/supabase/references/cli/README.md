# CLI

| Name | Description | Path |
|------|-------------|------|
| Supabase CLI: データベースコマンド | ローカルデータベースのスキーマ変更を差分として検出し、マイグレーション SQL を生成する。 | [db-commands.md](./db-commands.md) |
| Supabase CLI: Edge Functions コマンド | 新しい Edge Function を作成する。`supabase/functions/<name>/` ディレクトリにテンプレートファイルが生成される。 | [functions-commands.md](./functions-commands.md) |
| Supabase CLI: 型生成コマンド | データベーススキーマから TypeScript の型定義ファイルを生成する。 | [gen-commands.md](./gen-commands.md) |
| Supabase CLI: 検査コマンド | `supabase inspect db` コマンド群は、データベースのパフォーマンスや状態を診断するためのツール。 | [inspect-commands.md](./inspect-commands.md) |
| Supabase CLI: ローカル開発コマンド | プロジェクトを初期化し、`supabase/` ディレクトリと設定ファイルを生成する。 | [local-dev.md](./local-dev.md) |
| Supabase CLI: マイグレーションコマンド | 新しいマイグレーションファイルを作成する。タイムスタンプ付きの空の SQL ファイルが生成される。 | [migration-commands.md](./migration-commands.md) |
| Supabase CLI 概要 | npm / Homebrew / Scoop による CLI のインストール方法と認証設定、グローバルフラグ。 | [overview.md](./overview.md) |
| Supabase CLI: プロジェクト・その他コマンド | プロジェクト一覧を表示し、新規作成・削除などの管理コマンド。 | [project-commands.md](./project-commands.md) |
