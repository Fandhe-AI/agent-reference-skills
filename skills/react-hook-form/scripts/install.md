# Install

react-hook-form および関連パッケージのインストール。

## react-hook-form のインストール（npm）

```sh
npm install react-hook-form
```

## react-hook-form のインストール（yarn）

```sh
yarn add react-hook-form
```

## react-hook-form のインストール（pnpm）

```sh
pnpm add react-hook-form
```

## react-hook-form のインストール（bun）

```sh
bun add react-hook-form
```

## バリデーションリゾルバーのインストール（npm）

`@hookform/resolvers` は zod / yup / joi 等の外部バリデーションライブラリを接続するためのパッケージ。

```sh
npm install @hookform/resolvers
```

## バリデーションリゾルバーのインストール（yarn）

```sh
yarn add @hookform/resolvers
```

## バリデーションリゾルバーのインストール（pnpm）

```sh
pnpm add @hookform/resolvers
```

## バリデーションリゾルバーのインストール（bun）

```sh
bun add @hookform/resolvers
```

## zod と合わせてインストール（npm）

```sh
npm install react-hook-form zod @hookform/resolvers
```

## yup と合わせてインストール（npm）

```sh
npm install @hookform/resolvers yup
```

## DevTools のインストール（npm）

`@hookform/devtools` はフォームの状態をリアルタイムにデバッグするための開発ツール。開発依存として追加する。

```sh
npm install @hookform/devtools -D
```

## DevTools のインストール（yarn）

```sh
yarn add @hookform/devtools -D
```

## DevTools のインストール（pnpm）

```sh
pnpm add @hookform/devtools -D
```
