# vercel help

Display help information for Vercel CLI commands.

## Signature / Usage

```bash
vercel help
vercel help [command]
vercel --help
vercel [command] --help
```

## Notes

- Equivalent to the `--help` / `-h` global option on any command
- Shows available subcommands and options for the specified command
