# Contributing to rustc

The Rust project welcomes contributions to both the `rustc` compiler and its documentation.

## Primary Resource

The main guide for contributing to `rustc` is the **Rustc Dev Guide**:

- URL: https://rustc-dev-guide.rust-lang.org/
- Covers: compiler internals, architecture, how to build `rustc`, how to add tests, and how to submit changes.

## Contributing to This Book (The rustc Book)

The source for The rustc Book lives in the main Rust repository:

- Repository path: `src/doc/rustc`
- GitHub: https://github.com/rust-lang/rust/tree/HEAD/src/doc/rustc
- Direct edit link: https://github.com/rust-lang/rust/edit/main/src/doc/rustc/src/contributing.md

To contribute a fix or addition to the book, submit a pull request to the `rust-lang/rust` repository.

## Notes

- Most detailed guidance for contributing code to `rustc` itself is in the Rustc Dev Guide, not this page.
- The compiler and the book are both housed in the same `rust-lang/rust` monorepo.
