# Sandbox

Storybook 機能検証用のサンドボックスプロジェクト生成コマンド。

## サンドボックスプロジェクトの生成

```sh
storybook sandbox
```

## フレームワークを絞り込んで生成

```sh
storybook sandbox react
```

フィルター文字列に一致するフレームワークのみ表示する。

## 出力先ディレクトリを指定して生成

```sh
storybook sandbox --output ./sandbox-project
```

## Storybook を初期化せずにサンドボックスを生成

```sh
storybook sandbox --no-init
```

## バージョンを指定してサンドボックスを生成

```sh
storybook@8.3 sandbox
```
