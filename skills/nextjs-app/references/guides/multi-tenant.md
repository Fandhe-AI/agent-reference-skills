# Multi-tenant

Guidance for building a single Next.js application that serves multiple tenants with the App Router.

## Notes

- Next.js provides a reference architecture via the [Platforms Starter Kit](https://vercel.com/templates/next.js/platforms-starter-kit) example rather than a dedicated built-in API.
- No additional configuration options are documented; treat this as an architectural starting point rather than an API.
