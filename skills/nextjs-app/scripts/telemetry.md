# Telemetry

Next.js の匿名テレメトリ収集を切り替えるコマンド集。

## テレメトリを無効化する

```sh
next telemetry --disable
```

## テレメトリを有効化する

```sh
next telemetry --enable
```

## テレメトリコマンドのヘルプを表示する

```sh
next telemetry --help
```

Next.js が収集するテレメトリデータは完全に匿名。詳細は https://nextjs.org/telemetry を参照。

Source: https://nextjs.org/docs/app/api-reference/cli/next
