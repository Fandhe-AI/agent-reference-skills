# Introduction

The KiCad PCB Calculator is a set of utilities to help you find the values of components or other parameters of a layout.

## Tools Included

| Tool | Description |
|------|-------------|
| Regulators | Find resistor values for linear and low-dropout voltage regulators |
| Track Width | Calculate trace width for PCB conductors based on current and temperature rise |
| Electrical Spacing | Find minimum clearance between conductors for a given voltage |
| Trans Line | Model transmission line characteristics for RF and microwave engineering |
| RF Attenuators | Determine resistor values for PI, Tee, Bridged Tee, and Resistive Splitter attenuators |
| Color Code | Translate resistor color bands to resistance values |
| Board Classes | Reference IPC performance classes and PCB type definitions |
| E-Series | Identify standard E-series resistor combinations meeting a target resistance |
| Via Size | Calculate electrical and thermal properties of plated through-hole vias |

## Notes

- Access the PCB Calculator from the KiCad main window or from within the PCB Editor.
- Each tool operates independently; select the desired tool from the left-hand panel.
