# Welcome

KiCad is a free and open-source electronics design automation (EDA) suite. It supports schematic capture, circuit simulation, PCB layout, 3D rendering, and manufacturing output generation. KiCad runs on Linux, Windows, and macOS.

## What's New in KiCad 10.0

KiCad 10.0 introduces the following enhancements:

- Customizable toolbars
- Lasso selection modes
- Design imports from Allegro, PADS, and gEDA
- Design variants functionality
- Jumpered pins support
- Schematic grouping
- CSV pin definition handling
- Local power symbols
- Multiple symbol body styles
- Enhanced track length tuning with time-domain constraints
- Extended Design Blocks for PCB layouts
- Multi-layer (inner-layer) footprint graphics
- Unconstrained pin/gate swapping
- Graphical DRC rule editing
- Barcode generation
- Point objects for precise positioning
