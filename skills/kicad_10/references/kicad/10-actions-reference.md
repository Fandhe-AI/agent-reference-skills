# Actions Reference

Default hotkeys and action descriptions for the KiCad Project Manager.

## KiCad Project Manager

| Action | Default Hotkey | Description |
|--------|---------------|-------------|
| Archive Project… | — | Archive all project files into a ZIP archive |
| Browse Project Files | — | Open project directory in file explorer |
| Calculator Tools | — | Run component calculations, track width calculations, etc. |
| Clone Project from Repository… | — | Clone an existing remote Git repository as a new project |
| Close Project | — | Close the current project |
| Drawing Sheet Editor | `Ctrl+Y` | Edit drawing sheet borders and title block |
| Footprint Editor | `Ctrl+F` | Create, delete, and edit PCB footprints |
| Gerber Viewer | `Ctrl+G` | Preview Gerber output files |
| Image Converter | `Ctrl+B` | Convert bitmap images to schematic or PCB components |
| Launch preferred text editor | — | Open the configured text editor |
| Local History | — | Display or conceal the local history panel |
| New Jobset File… | — | Create a new jobset file |
| New Project… | `Ctrl+N` | Create a new project (optionally from a template) |
| Open Demo Project… | — | Open a demonstration project |
| Open Jobset File… | — | Open an existing jobset file |
| Open Project… | `Ctrl+O` | Open an existing KiCad project |
| PCB Editor | `Ctrl+P` | Edit the PCB in the PCB editor |
| Plugin and Content Manager | `Ctrl+M` | Run the Plugin and Content Manager |
| Restore Project from Local History… | — | Recover project files from the backup history |
| Schematic Editor | `Ctrl+E` | Edit schematic in the schematic editor |
| Symbol Editor | `Ctrl+L` | Create, delete, and edit schematic symbols |
| Unarchive Project… | — | Extract project files from a ZIP archive |

## Notes

- Actions without a default hotkey (`—`) can be assigned custom shortcuts via **Preferences → Preferences… → Hotkeys**
- Hotkeys shown are defaults and may differ if customized
