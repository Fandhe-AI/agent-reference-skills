# Properties Editor

The Properties Editor is a right-side panel that appears when an item is selected on the canvas.

## Tabs

| Tab | Description |
|-----|-------------|
| Item Properties | Fields specific to the selected item type (line, rectangle, text, bitmap) |
| General Options | Default properties and margin settings for the sheet |

## Notes

- The Properties Editor is only visible when an item is selected
- Changes are **not applied** until the **Apply** button is clicked
- Fields in the Item Properties tab vary depending on the selected item type
- Works in conjunction with the Design Inspector for managing items
