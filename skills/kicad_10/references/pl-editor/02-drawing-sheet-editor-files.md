# Drawing Sheet Editor Files

The Drawing Sheet Editor reads and writes drawing sheet files with the `.kicad_wks` extension.

## Usage

- On first launch, the default KiCad drawing sheet is loaded until a different file is opened.
- A custom drawing sheet can be selected per-project in the **Page Setup** dialog of the Schematic or PCB Editor.

## Notes

- File extension: `.kicad_wks`
- Custom sheets are portable and reusable across projects and organizations
